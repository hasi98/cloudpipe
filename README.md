# CloudPipe 🚀

> Download anything to your Google Drive instantly

CloudPipe is a cloud download manager that fetches 
torrents, magnet links and direct URLs on remote 
servers and sends them straight to your Google Drive.


## ✨ Features

- 🔗 Works with magnet links, .torrent files, HTTP/FTP
- ☁️ Files go straight to YOUR Google Drive
- ⬇️ Or download directly from our server
- 🔒 Your IP stays completely hidden
- 📝 Zero logs policy
- 🌐 No software to install — works in browser
- ⚡ High-speed cloud servers
- 🆓 Free plan available

## 🛠️ Tech Stack

- **Frontend:** Next.js 14 (App Router) + Tailwind CSS
- **Auth + Database:** Supabase
- **Download Engine:** aria2c
- **Worker:** Node.js
- **CDN + DDoS:** Cloudflare
- **Cloud Storage:** Google Drive API
- **Payments:** Creem
- **Hosting:** OVH VPS + Oracle Cloud Free Tier
- **Deploy:** Netlify + GitHub Actions
